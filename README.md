I'm a first-year HCI student at Universität Duisburg-Essen, studying how people interact with technology — and how we can make that interaction better. I'm at the beginning of my programming journey, but I care a lot about building things that are actually useful: structured, readable, and built with the person in mind.

## 🌟 Featured Project

<div align="center">

### 📂 pdf-reading-tracker

</div>

> An AI-powered document processing system designed to turn your reading pile into structured, searchable knowledge.

**What it does:**
- Accepts **PDF, TXT, and DOCX** files as input
- Extracts and processes text content automatically
- Uses an **AI model to summarize** each document
- **Tags and categorizes** documents by topic
- Saves all results into clean, structured **CSV files** for easy review and searching

**Why I built it:** I wanted a system that could handle large reading loads and turn raw documents into usable knowledge — the kind of tool I'd actually use myself.

**Tech:** `Python` · `AI API integration` · `file parsing` · `CSV output`
